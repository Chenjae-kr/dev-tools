# UI_CONVENTIONS.md

Dev Tools UI 통일 가이드 (v1)

## 1) 레이아웃 템플릿
- Form + Result
- Form + Preview
- Dual Input + Result

## 2) 섹션 타이틀 언어
- 섹션/패널 타이틀은 영문 대문자 스타일로 통일:
  - INPUT / SETTINGS / OUTPUT / PREVIEW / RESULT / EXPORT

## 3) 공통 클래스 우선
- 인라인 스타일 대신 공통 유틸 클래스를 사용
  - `row-tight`, `mt-8`
  - `input-xs`, `input-sm`, `input-md`
  - `color-input`

## 4) 버튼 카피
- 생성: `⚡ 생성`
- 변환: `⚡ 변환`
- 복사: `COPY`
- 다운로드: `⬇ 다운로드`

## 5) 카드 규칙
- 설명: 1문장
- 태그: 최대 3개 + `+N`
- 입출력 미리보기: hover 시 노출
